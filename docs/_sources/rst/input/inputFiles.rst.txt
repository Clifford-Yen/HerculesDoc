===========
Input Files
===========

Hercules take different kinds of input files depending on the type of analysis you want to perform. Regardless of the type of analysis, Hercules always requires a parameter input file (default: ``parameters.in``) and a source input file (default: ``source.in``). This section will guide you through the different types of input files that Hercules uses.
